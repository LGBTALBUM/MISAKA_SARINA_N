Overview

Data flow

Generated VocaDB data lives in src/data/music.ts. Manual corrections are kept in src/data/musicOverrides.ts so sync operations can be repeated without overwriting curated site decisions.

Visibility rules

The catalogue prioritizes main releases while moving instrumental, off-vocal, karaoke, demo, and profile entries into lower-priority sections. Overrides can force entries into main or secondary sections when the automated rules are not enough.
